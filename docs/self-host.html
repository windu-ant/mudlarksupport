Self-Host Your Proxy

Prefer to run your own infrastructure? You can host your own MUDlark proxy server.

Windows Linux macOS Open Source

Host It Yourself

MUDlark supports an optional proxy relay so your sessions can stay active while iOS backgrounds the app. If you prefer full control, you can run your own proxy server instead of using the default hosted one.

The public setup guide includes installation and configuration instructions for Windows, Linux, and macOS.

Open the self-hosting setup repository

Quick Start

1. Open the setup repository linked above
2. Follow the install guide for your platform (Windows, Linux, or macOS)
3. Run your proxy server and verify it is reachable
4. Configure MUDlark to use your server endpoint

Notes

  • Self-hosting is optional. You can still use MUDlark without hosting your own server.
  • Running your own proxy gives you control over deployment, updates, and infrastructure.
  • Review the repository documentation for platform-specific dependencies and security recommendations.
